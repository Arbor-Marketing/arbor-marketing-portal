# ARBOR Marketing Portal

Marketing Calendar & Flyer Generator for ARBOR Financial Group

## Project Contents

- **calendar.html** - ARBOR Marketing Calendar 2026 (Interactive React-based calendar)
- **flyer-generator.html** - ARBOR Flyer Generator (LO Flyer Pack & Open House Flyer tools)

## Deployment

This project is deployed on Vercel. To deploy:

1. Push changes to GitHub
2. Vercel will automatically build and deploy

## Files

- `calendar.html` - Full marketing calendar with campaigns, pillars, and deliverables
- `flyer-generator.html` - Flyer generation tool with profile management and print functionality

## Local Development

Simply open the HTML files in your browser to test locally.

## Vercel Deployment

- Connect this repository to Vercel
- Set root directory (if needed)
- Deploy!
